\# Flying Bird



Unity 2D arcade game inspired by Flappy Bird.



\## Play Online



https://lebedinecslavik5-png.github.io/FlyingBird/



\## Features



\- Smooth bird movement and rotation

\- Random obstacle spawning

\- Score and high score system

\- Sound on/off toggle

\- Game Over and restart system

\- WebGL browser build

\- Mobile-style portrait gameplay



\## Controls



\- Left Mouse Button

\- Space



\## Built With



\- Unity

\- C#

\- Unity Input System

\- WebGL
